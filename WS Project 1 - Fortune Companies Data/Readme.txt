**🌐 Web Scraping Project: Fortune 500 Companies Data**  

A Python script that **extracts and organizes** the list of largest US companies by revenue from
Wikipedia using web scraping and pandas.  

**📦 Libraries Used:**

- **BeautifulSoup** → Parses HTML to extract table data.  
- **requests** → Fetches the webpage content.  
- **pandas** → Structures scraped data into a clean DataFrame.  

**🚀 Key Steps:**

1. **Scraped** the Wikipedia page for the companies' table.  
2. **Extracted headers** (`th`) and row data (`td`).  
3. **Structured data** into a pandas DataFrame.  
4. **Exported** for further analysis (CSV/Excel).  

**💡 Why It’s Cool?**

✅ **Automates data collection** (no manual copying!).  
✅ **Flexible** (adaptable to other Wikipedia tables).  
✅ **Pandas-ready** for instant analysis.
