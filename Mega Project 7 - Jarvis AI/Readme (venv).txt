>> STEPS TO PERFORM BEFORE RUNNING THE PROGRAM <<

Before running the program, must install the required modules mentioned in the requirements.txt folder. To install them, follow the given instructions...

1. Activate your virtual environment (assuming the venv is named venv):
venv\Scripts\activate

2. Run the command to install all dependencies:
pip install -r requirements.txt

This will read the requirements.txt file and install all the packages listed, along with their specified
versions. After this, you'll be able to run the program (i.e. Jarvis AI) and will not get any error!
